Text Tags

HTML provides various tags to structure and format text:

  • <h1> to <h6> - Headings
  • <p> - Paragraphs
  • <em> - Emphasized text
  • <strong> - Strong text
  • <mark> - Highlighted text

Example:

This is a Heading

This is a paragraph of text.

This text is emphasized.

This text is strong.

This text is highlighted.

Image Tags

HTML tags for adding and manipulating images:

  • <img> - Insert an image
  • <figure> - Container for media content
  • <figcaption> - Caption for media content

Link Tags

HTML tags for creating links:

  • <a> - Anchor tag for links
  • <href> - Specify the URL
  • <target> - Specify where to open the link

List Tags

HTML tags for creating lists:

  • <ul> - Unordered list
  • <ol> - Ordered list
  • <li> - List item
  • <dl> - Description list
  • <dt> - Description term
  • <dd> - Description definition

Example:

  • Item 1
  • Item 2
  • Item 3
  1. First item
  2. Second item
  3. Third item
